# How to run it?
* Put your API key in .env file
```declarative
source .env
```
* execute run.sh script
```declarative
./run.sh
```

## Configuration
* Configuratoins
```declarative
GOOGLE_GENAI_USE_VERTEXAI=FALSE
GOOGLE_API_KEY="Your API Key"
FOOTBALL_API_KEY
```
FOOTBALL_API_KEY can be obtained from https://www.api-football.com/
